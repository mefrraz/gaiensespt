## 2. Fix Hora em Jogos com Resultado

### Problema
O parser em `fpbApi.ts` só extrai `hora` quando `status === 'AGENDADO'`. Para jogos FINALIZADOS, a hora fica vazia (`""`), mas o Game.tsx mostra `--:--` ou `00:00`.

O HTML da FPB para jogos finalizados mostra a pontuação no `.hour h3` em vez da hora original:
```
<div class="hour align-self-center"><h3>78H65</h3></div>  ← impossível separar hora de score
```

### Solução
- **Game.tsx:** Mostrar a hora APENAS quando `match.hora` tem valor e o jogo NÃO está finalizado (ou está AO VIVO)
- Para jogos FINALIZADOS, mostrar apenas a data (sem placeholder de hora)
- Para jogos AGENDADOS, mostrar data + hora

| Estado | Mostrar Data | Mostrar Hora |
|--------|-------------|--------------|
| FINALIZADO | ✅ | ❌ |
| A DECORRER | ✅ | ✅ |
| AGENDADO | ✅ | ✅ |

### Ficheiros a alterar
- `web/src/pages/Game.tsx` — condicionar exibição da hora a `match.hora && !isFinished` e `match.hora && isLive`
- `web/src/pages/Games.tsx` (GameCard) — mesma lógica: hora só para não-finalizados
