# Plano: Game Page — Fix Hora Duplicada + Últimos Jogos

## 1. Fix Hora (mostrada 3x)

### Problema
Na ficha de jogo de um jogo AGENDADO, a hora aparece **3 vezes**:
1. Abaixo do **VS** no hero (junto com "VS" e hora)
2. Na linha **data + hora** abaixo dos teams no hero
3. No **card Data** no fundo

### Solução
Mostrar hora APENAS no **card Data** no fundo. Remover dos outros 2 locais:

**Hero — remover hora abaixo de VS:**
```tsx
// ANTES (linha 135-140)
{hasHora && (
    <span className="text-xs font-mono font-bold text-zinc-700 dark:text-zinc-300">
        {match.hora!.slice(0, 5)}
    </span>
)}

// DEPOIS: remover bloco inteiro
```

**Hero — data/hora (linhas 146-153):**
```tsx
// ANTES
<div className="mt-6 flex items-center justify-center gap-2 ...">
    <span className="capitalize">{dateFormatted}</span>
    {hasHora && (
        <>
            <span className="w-1 h-1 rounded-full bg-zinc-300" />
            <span className="font-mono">{match.hora!.slice(0, 5)}</span>
        </>
    )}
</div>

// DEPOIS: mostrar SÓ a data, sem hora
<div className="mt-6 flex items-center justify-center text-xs text-zinc-500">
    <span className="capitalize">{dateFormatted}</span>
</div>
```

**Card Data (manter como está):**
```tsx
// MANTER — este é o único local com a hora
{hasHora && (
    <p className="text-sm text-zinc-500 font-mono">{match.hora!.slice(0, 5)}</p>
)}
```

### Resultado

```
┌─────────────────────────────────────┐
│  🗓 Data                            │
│  domingo, 17 de maio de 2026        │
│  15:30                              │  ← hora aqui
└─────────────────────────────────────┘
```

## 2. "Últimos Jogos" — Mesmo Escalão

### O quê
Mostrar os últimos 3 jogos **FINALIZADOS** do FC Gaia no **mesmo escalão** do jogo que está a ser visto.

### Exemplo visual
```
┌─────────────────────────────────────┐
│  Últimos Jogos — Sénior Masculino    │
│                                     │
│  🟢 FC GAIA 78 - 65 OPONENTE 15 Mai │
│  🔴 FC GAIA 62 - 70 OPONENTE 10 Mai │
│  🟢 FC GAIA 88 - 52 OPONENTE 04 Mai │
└─────────────────────────────────────┘
```

### Como obter os dados
Já temos todos os jogos no Supabase (`games_2025_2026`). Basta um `useEffect` que faz:

```typescript
supabase
    .from('games_2025_2026')
    .select('*')
    .or(`equipa_casa.ilike.%gaia%,equipa_fora.ilike.%gaia%`)
    .eq('escalao', match.escalao)
    .eq('status', 'FINALIZADO')
    .order('data', { ascending: false })
    .limit(3)
```

### Formato
- Indicador verde (V) / vermelho (D) como dot
- Nome do adversário (a equipa que NÃO é FC Gaia)
- Resultado (scores do FC Gaia e adversário)
- Data (dia mês abreviado)
